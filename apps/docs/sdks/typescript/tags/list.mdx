## Response

<ResponseField name="id" type="string" required>
  The unique ID of the tag.
</ResponseField>

<ResponseField name="name" type="string" required>
  The name of the tag.
</ResponseField>

<ResponseField name="color" type="string" required>
  The color of the tag.
</ResponseField>

<RequestExample>

```ts
await dub.tags.list({});
```

</RequestExample>

```ts
[

```

</ResponseExample>
