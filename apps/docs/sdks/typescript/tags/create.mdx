## Arguments

<ParamField body="tag" type="string" required>
  The name of the tag to create.
</ParamField>

<ParamField body="color" type="string">
  The color of the tag. If not provided, a random color will be used from the
  list: `red`, `yellow`, `green`, `blue`, `purple`, `pink`, `brown`.
</ParamField>

## Response

<ResponseField name="id" type="string" required>
  The unique ID of the tag.
</ResponseField>

<ResponseField name="name" type="string" required>
  The name of the tag.
</ResponseField>

<ResponseField name="color" type="string" required>
  The color of the tag.
</ResponseField>

<RequestExample>

```ts
await dub.tags.create({
  tag: "news",
});
```

</RequestExample>

<ResponseExample>

```ts
{
  id: "clv3j1h7o00032u1ngo8luv84",
  name: "news",
  color: "blue"
}
```

</ResponseExample>
