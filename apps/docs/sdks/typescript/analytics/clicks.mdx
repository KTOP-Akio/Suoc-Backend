## Response

The number of clicks matching the specified queries.

<RequestExample>

```typescript
await dub.analytics.clicks({
  key: "dummy",
  domain: "dub.sh",
  interval: "30d",
});
```

</RequestExample>

<ResponseExample>

```typescript
20
```

</ResponseExample>
