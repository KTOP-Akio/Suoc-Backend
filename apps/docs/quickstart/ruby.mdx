## 2. Install

To install the Dub Ruby SDK, run the following command:

```bash RubyGems
gem install dub
```

Initialize the Dub Ruby SDK by creating a new instance of the `Dub` class.

```ruby
require 'dub'

d = ::OpenApiSDK::Dub.new(
  workspace_id: "ws_clugls3...",
)

d.config_security(
  ::OpenApiSDK::Shared::Security.new(
    token: "DUB_API_KEY",
  )
)
```

## 3. Create link

Let's create a short link using the Dub Node.js SDK.

```ruby index.rb
res = d.links.create(workspace_id="<value>", project_slug="<value>", request_body=::OpenApiSDK::Operations::CreateLinkRequestBody.new(
    url: "https://google/com",
    external_id: "123456",
    tag_ids: [
    "<value>",
  ],
  ))
```
