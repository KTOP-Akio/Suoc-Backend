# `dub-cli`

a cli for easily shortening URLs.

## Running Locally

To install the package and then,

1. Navigate to the cli folder

   ```bash
   cd packages/cli
   ```

2. Start the development server

   ```bash
   pnpm dev
   ```

3. Run commands

   ```bash
   pnpm start [command]
   ```

4. See all available commands and options

   ```bash
   pnpm start help
   ```
